Developer quickstart

Commands:

Restore packages:
- dotnet restore

Build solution:
- dotnet build

Run tests:
- dotnet test

Run web project locally:
- dotnet run --project HomeReadingLibraryWeb

Notes:
- Project targets .NET 10. Ensure SDK 10 is installed.
- Frontend uses Angular 21; see ClientApp in HomeReadingLibraryWeb for the Angular app.
